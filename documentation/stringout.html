stringout Records

Normal Operation

The variable x stands for the written or read value.

DOUBLE format (e.g. %f):
Not allowed.
LONG format (e.g. %i):
Not allowed.
ENUM format (e.g. %{):
Not allowed.
STRING format (e.g. %s):
Output: x=VAL
Input: VAL=x

Initialization

During initialization, the @init handler is executed, if present. All format converters work like in normal operation.
